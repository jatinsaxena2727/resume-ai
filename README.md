# 🚀 Resume AI Analyzer

A smart web-based Resume Analyzer built using **Flask + Python** that compares a candidate’s resume with a job description and generates an AI-based matching score.

---

## 🔥 Features

- 📄 Upload Resume (PDF supported)
- 🧠 Extracts text from resume
- 📊 Compares with Job Description
- 🎯 Generates ATS-like score (0–100)
- 📑 Resume preview display
- 🌙 Modern dark UI
- ⚡ Fast Flask backend

---

## 🛠️ Tech Stack

- Python 🐍  
- Flask 🌐  
- HTML / CSS 🎨  
- PyPDF2 📄
